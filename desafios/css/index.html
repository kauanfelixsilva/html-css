Estilos CSS

Css Inline

O estilo inline é uma forma de aplicar o CSS diretamente dentro da própria tag HTML, usando o atributo style.

Estilos Internos

O estilo interno é uma forma de aplicar o CSS diretamente dentro do próprio arquivo HTML, mas separado das tags, usando a tag style dentro do head.

Estilos Externos

O estilo externo é quando você cria um arquivo separado só para o CSS, geralmente com a extensão .css, e conecta esse arquivo ao seu HTML usando a tag dentro do .
